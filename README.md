# 开源软件指南

## 概览、历史和知识产权

自由软件（free software）: 赋予用户运行、复制、分发、学习、修改并改进软件等自由的软件产品

> GNU（GUN is Not Unix）计划

开源软件的十条准则：
1. 别人分发你的软件，你不能问他要钱
2. 必须有源代码
3. 允许他人修改你的作品
4. 可以要求保持旧版本源码不变
5. 不歧视个人或团队
6. 不歧视行业或领域
7. 原作者的许可不能被修改
8. 许可不能针对具体产品
9. 许可不能对其他产品做限制
10. 不能限制特定的技术 

主流的许可证（license）协议：
- 宽松式
  - BSD
  - MIT
  - Apache 2.0

产权：法律上对财产权的裁判
版权（著作权）：指作者或其他人（包括法人）依法对某一著作物享有的权利

声明版权示例: __copyright @2020 xxx, all rights reserved.__
